Partiendo del algoritmo de Ignacio y Axel:

Conocemos que:

  • Se debe ingresar o pedir al usuario, la forma Polinómica de Grado 3, teniendo en cuenta que dichos valores deben almacenarse en variables y que puede que algunas no se ingresen, por lo tanto, si se piden todas las variables al usuario, se le debe informar que rellene con cero la/s variable/s que no se ingrese/n.
  • Una vez ingresados los datos, el algoritmo indica que se debe derivar la función.
  • Luego se debe hallar la recta tangente en un punto especifico de la derivada y calcular su raíz.
  • Una vez calculada la raíz, se verifica en la función si es raíz o si se aproxima a cero, si es raíz termina el algoritmo.
  • Si no es raíz, se repite el procedimiento de buscar la recta tangente en ese valor de raíz que se caculó y se halla su raíz, y se la evalua nnuevamente en la función para ver si es cero.
  • El algoritmo debe tener la decisión de parar la iteración llegado un número de decimales, opción que se podría pedir al usuario.

Pasos

  1. Derivar f(x)
  2. Definir un valor inicial i.
  3. Hallar Tangente: \[T:f'(i)(x-i)+f(i)\]
  4. Hallar raíz de T.
  5. Evaluar la raíz hallada en la función para comprobar si es raíz, sino, la raíz se convierte en un nuevo valor i.

Ahora que ya manejamos la introdución de datos

Pasamos a realizar las operaciones. Para ello crearemos funciones pequeñas que sólo realizan lo que haríamos en una hoja, evaluar x en la funcion, su derivada, calcular la ordenada al origen de la recta tangente e y finalmente aplicamos Newton-Raphson para hallar la raíz de la recta tangente que nos dará un nuevo x.

\[x_{n+1} = \frac{-b}{f'(x)}\]


def f(x):
    return a*x**3 + b*x**2 + c*x + d

def derivada(x):
    return 3*a*x**2 + 2*b*x + c

def ordenada(x):
    return f(x) - derivada(x) * x

def raiz(x_ini, max_i):
  import time
  x=x_ini
  for _ in range(max_i):
    if derivada(x) == 0:
      print("¡Derivada igual a cero! Reintentar con otro x inicial")
      return None
    x = -ordenada(x) / derivada(x)
    print(x)
    time.sleep(1)
  return x


newton=raiz(n,i)
print("\n")
display(Math(fr"\text{{La raíz es aproximadamente: }} \textbf{{x}} \approx {newton} \text{{, en }} {i} \text{{ iteraciones.}}"))

Mostrar una Tabla:

Con las iteraciones, se agrega otro método de la clase display: clear_output se usa time.sleep(secs) para que vayan mostrandose las iteracciones cada un segundo. Se usa LaTeX para el armado de la tabla mediante un \begin{array}.


from IPython.display import display, Math, clear_output
import time

def raiz(x_ini, max_i):
    x = x_ini
    # Encabezado de la tabla (solo se muestra una vez)
    tabla = r"""
    \begin{array}{|c|c|c|c|c|c|}
    \hline
    \text{N° Iteración} & x & f(x) & \textbf{Pendiente en }(x, f(x) & \textbf{Ordenada de la tangente} & \textbf{Raíz de la tangente} \\
    \hline
    """
    display(Math(tabla + r"\hline \end{array}"))  # Tabla inicial vacía

    for _ in range(max_i):
        dx = derivada(x)
        if dx == 0:
            clear_output(wait=True)
            display(Math(r"\text{¡Error! Derivada cero.}"))
            return None

        x_nuevo = -ordenada(x) / dx
        tabla += fr"{_ + 1} & {x:} & {f(x):} & {dx:} & {ordenada(x):} & {x_nuevo:} \\ \hline "

        clear_output(wait=True)  # ¡Borra la salida anterior!
        display(Math(tabla + r"\end{array}"))  # Muestra la tabla actualizada
        time.sleep(1)  # Pausa de 1 segundo
        x = x_nuevo

    return x


newton=raiz(n,i)
print("\n")
display(Math(fr"\text{{La raíz es aproximadamente: }} \textbf{{x}} \approx {newton} \text{{, en }} {i} \text{{ iteraciones.}}"))
